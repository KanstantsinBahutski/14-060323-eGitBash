 1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

Для начала нужно создать новую пустую папку компьютере, в которую мы хотим скопировать репозиторий.
Далее нужно открыть командную строку или терминал на компьютере и перейти в папку, которую мы создали.
Это можно сделать с помощью команды cd, указав путь к папке. 
Теперь мы можем скопировать репозиторий на свой компьютер с помощью команды git clone.
Для этого нам понадобится URL репозитория на GitHub. 

2. Как получить на локальный компьютер изменения с github.com

Использовать команду git pull

3. Какую информацию нам дают команды :
git status
git log
git remote -v 

git status: Эта команда показывает текущее состояние вашего рабочего каталога.
Она покажет, какие файлы были изменены, добавлены или удалены. 
Это полезно для проверки статуса вашей локальной копии относительно удаленного репозитория.

git log: Эта команда показывает историю коммитов в репозитории. 
Она покажет информацию о каждом коммите, включая автора, дату и время коммита, и сообщение коммита. 

git remote -v: Эта команда покажет список удаленных репозиториев, связанных с вашей локальной копией репозитория,
 а также URL-адреса этих репозиториев. 
Она выводит два столбца: "origin" (основной удаленный репозиторий) и 
"fetch/push" (URL-адрес удаленного репозитория для извлечения/отправки изменений).
